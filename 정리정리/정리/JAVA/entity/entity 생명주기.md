## Entity 생명주기

JPA에서 엔티티는 다음 4가지 상태를 가집니다:

1. **Transient (비영속)**:
    - 영속성 컨텍스트와 관계없는 상태.
    - 데이터베이스와 연관되지 않은 상태.
2. **Persistent (영속)**:    
    - 영속성 컨텍스트에 관리되는 상태.
    - 데이터베이스와 연동 가능.
3. **Detached (준영속)**:
    - 영속성 컨텍스트에서 분리된 상태.
    - 데이터베이스와의 동기화가 더 이상 이루어지지 않음.
4. **Removed (삭제)**:
    - 삭제가 예약된 상태.
    - 데이터베이스에서 삭제됨.
