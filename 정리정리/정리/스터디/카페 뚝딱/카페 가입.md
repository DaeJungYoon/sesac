1. 카페 내에서 사용할 닉네임
    - 중복되면 안된다 (unique)
    - 필수 입력값 (notblank)
2. 프로필 이미지
    - 필수X
3. 카페 규칙을 지킬지 확인
    - 라디오 버튼 (네 / 아니요)
    - 아니요를 누르면 카페 가입버튼 비활성화
    - 필수 입력값
4. 개인정보 공개여부 설정
    - 카페 운영진에게 자신의 정보를
       보여줄지 확인 받기
    - 라디오 버튼 (허용 / 비허용)
       (개인정보는 비공개할 수도 있음)
    - 필수 입력값
5. 카페 가입 시 각오 한 마디
    - 필수 입력값 (NotBlank)
